Welcome to the EUS Website

What is EUS?

EUS is a public screenshot server that I made because I like running my own stuff for my own use.

Setup for different applications is listed on the Downloads page

EUS is built on top of my nodejs based server Revolution

The EUS Website, EUS and Revolution are all open source on github

The font used on the site is Saira Semi Condensed
